# Public Assets

This directory contains static assets served by Next.js.

## Required Files

### `demo.mp4`
Place your demo video file here for the "See It In Action" section on the homepage.

**Requirements:**
- Format: MP4 (H.264)
- Aspect Ratio: 16:9
- Recommended Resolution: 1920x1080 or 1280x720
- Duration: 10-30 seconds
- File size: < 10MB for optimal loading

**Where to get a demo video:**
1. Use your own sample edited footage
2. Download stock footage from [Pexels](https://www.pexels.com/videos/) or [Pixabay](https://pixabay.com/videos/)
3. Use a short clip from any video editing project

**Example sources:**
- Cinematic drone shots
- Time-lapse sequences
- Product demos
- Behind-the-scenes footage

Once you add `demo.mp4` to this folder, it will automatically play in the Editor Showcase section on the homepage.
